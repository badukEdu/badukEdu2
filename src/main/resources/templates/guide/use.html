상품소개

/guid/product

관리자가 등록한 상품이 소개되는 페이지

교육자마당

숙제 생성/조회

교육자마당 클릭 -> 숙제 생성/조회 클릭 -> 숙제 정보 입력 후 등록하기 버튼

하면 아래 리스트에 등록됨

숙제 배포

사이드탭의 숙제 배포 클릭 후 학습그룹 선택 -> 학습그룹에 등록된 숙제들과 학습자들 출력

보낼 숙제들과 멤버들 선택 후 전송하기 버튼 누르면 각 학습자들에게 배포되면서 등록된 이메일에 전송

숙제 학습 진도 조회

사이드탭 숙제 학습 진도 조회 클릭 후 -> 출력된 숙제를 더블클릭

배포된 숙제를 학생들이 제출한 내용을 보임


질문 사항 내용을 클릭하면 학생들이 질문한 내용에 답변을 해줄 수도 있고,
오른쪽에 있는 평가 항목에서 일괄로 평가 처리할 수 있음

학습서비스

숙제 목록

학습서비스 탭에 숙제목록을 클릭하면 배포된 숙제를 확인할 수 있습니다.

숙제 제출

작성 버튼 클릭하면 작성하는 폼으로, 내용확인을 누르면 숙제 내용과 답변, 질문, 선생님의 답변까지 다 볼 수 있습니다
